| Key     | Description                                                  |
| ------- | ------------------------------------------------------------ |
| C-w     | Add word at point to search string                           |
| C-M-y   | Add character at point to search string                      |
| M-s C-e | Add rest of line at point to search string (Emacs 24.4 or later) |
